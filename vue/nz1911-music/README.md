### 命名规范
1. 文件夹的命名 小写字母连接符的命名方式  my-demo 
2. 文件的命名   首字母小写字母 小驼峰 或者也是用连接符  my-demo.js  myDemo.js
3. 组件和类文件和文件夹  大驼峰    MyBanner.vue   MyBanner.js 
4. index.js  index.vue  
   在模板中  -+小写字母 = 大写字母  less-demo = lessDemo
